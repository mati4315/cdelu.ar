## 🔍 Posibles Problemas y Soluciones

### Problema 1: API Key no configurada
**Síntomas:** HTTP 401
**Solución:**
```bash
# Verificar .env
cat .env | grep API_KEY
# Debe mostrar: API_KEY=EbVOmsJC5rVywOGVLQyvkmUBLhBtVASY
```

### Problema 2: Middleware faltante
**Síntomas:** HTTP 401
**Solución:** Añadir middleware en `src/middlewares/auth.js`:

```javascript
const authenticateApiKey = (request, reply) => {
  const config = require('../config/default');
  const apiKey = request.headers['x-api-key'];

  if (!apiKey || apiKey !== config.apiKey) {
    return reply.status(401).send({ error: 'API Key inválida' });
  }
};
```

### Problema 3: Middleware no aplicado
**Síntomas:** HTTP 401
**Solución:** Aplicar middleware en la ruta:

```javascript
// En src/features/news/news.routes.js
fastify.post('/api/v1/news/external', {
  onRequest: [authenticateApiKey],
  // ... resto de configuración
}, ctrl.createExternalNews);
```

### Problema 4: Servidor no reiniciado
**Síntomas:** Cambios no aplican
**Solución:**
```bash
# Detener servidor (Ctrl+C)
# Reiniciar
npm start
```
