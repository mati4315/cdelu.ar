# 🔄 MÓDULO DE INTEGRACIÓN: WordPress → CdelU API

## 📋 Información General

**Nombre del Módulo:** WordPress-CdelU Sync
**Versión:** 1.0.0
**Fecha de Creación:** Marzo 2026
**Propósito:** Sincronización automática de publicaciones desde WordPress hacia CdelU

---

## 🎯 Propósito y Alcance

### **Objetivo Principal**
Permitir que publicaciones creadas en WordPress se sincronicen automáticamente con el sistema CdelU, manteniendo ambos sistemas actualizados en tiempo real.

### **Casos de Uso**
- ✅ **Publicación automática:** Al publicar en WordPress, se crea automáticamente en CdelU
- ✅ **Sincronización bidireccional:** Futura expansión para sincronización de vuelta
- ✅ **Multi-sitio:** Soporte para múltiples instancias de WordPress
- ✅ **Categorización:** Mapeo automático de categorías entre sistemas

### **Beneficios**
- 🔄 **Eliminación de duplicación manual** de contenido
- 📊 **Centralización** de contenido en CdelU
- 🤖 **Automatización** de procesos editoriales
- 📱 **Compatibilidad móvil** inherente

## 🔄 Flujo de Datos

### **Secuencia de Sincronización**

```
1. 📝 Usuario crea/edita post en WordPress
2. 🎣 Hook 'save_post' se activa
3. 🔍 Plugin valida el post (tipo, estado, etc.)
4. 📦 Extrae datos del post (título, contenido, imagen, etc.)
5. 🔧 Procesa contenido (limpia HTML, limita longitud)
6. 🌐 Envía HTTP POST a CdelU API
7. 🔐 CdelU valida API Key
8. 💾 Inserta noticia en base de datos
9. 🔗 Actualiza feed unificado (trigger automático)
10. ✅ Respuesta de éxito/error a WordPress
11. 📝 Registra resultado en logs
```

### **Campos Mapeados**

| WordPress | CdelU API | Tipo | Requerido | Notas |
|-----------|-----------|------|-----------|-------|
| post_title | titulo | string | ✅ | Título de la noticia |
| post_content | descripcion | string | ✅ | Contenido principal |
| _thumbnail_id | image_url | string | ❌ | URL de la imagen (opcional) |
| permalink | original_url | string | ❌ | Link original a WordPress (opcional) |

**🎯 Simplificación:** Solo envía `titulo` y `descripcion`. El servidor configura automáticamente:
- ✅ `is_oficial = 1` (siempre oficial)
- ✅ `created_by = 1` (usuario Matias Moreiraa)
- ✅ `published_at = NOW()` (fecha/hora actual)
- ✅ `resumen = generado automáticamente` (si IA está disponible)

## 🔐 Sistema de Autenticación

### **API Key Authentication**
- **Tipo:** Header-based authentication
- **Header:** `X-API-Key`
- **Valor:** `EbVOmsJC5rVywOGVLQyvkmUBLhBtVASY`
- **Validación:** Comparación exacta en middleware

### **Implementación en WordPress**
```php
$headers = array(
    'Content-Type' => 'application/json',
    'X-API-Key' => 'EbVOmsJC5rVywOGVLQyvkmUBLhBtVASY'
);
```

### **Implementación en CdelU**
```javascript
const authenticateApiKey = (request, reply) => {
    const config = require('../config/default');
    const apiKey = request.headers['x-api-key'];

    if (!apiKey || apiKey !== config.apiKey) {
        return reply.status(401).send({ error: 'API Key inválida' });
    }
};
```

```php
// En functions.php o plugin - VERSIÓN SIMPLE
define('CDELU_API_URL', 'http://localhost:3001/api/v1/news/external');
define('CDELU_API_KEY', 'EbVOmsJC5rVywOGVLQyvkmUBLhBtVASY');

// Función simple para sincronizar
function sync_post_to_cdelu($post_id) {
    $post = get_post($post_id);
    
    if (!$post || $post->post_status !== 'publish') {
        return; // Solo publicados
    }
    
    $data = array(
        'titulo' => $post->post_title,
        'descripcion' => wp_strip_all_tags($post->post_content),
        'image_url' => get_the_post_thumbnail_url($post_id),
        'original_url' => get_permalink($post_id)
    );
    
    $response = wp_remote_post(CDELU_API_URL, array(
        'headers' => array(
            'Content-Type' => 'application/json',
            'X-API-Key' => CDELU_API_KEY
        ),
        'body' => json_encode($data)
    ));
    
    if (is_wp_error($response)) {
        error_log('CdelU sync error: ' . $response->get_error_message());
    } else {
        $code = wp_remote_retrieve_response_code($response);
        if ($code === 201) {
            update_post_meta($post_id, '_cdelu_synced', '1');
        }
    }
}

// Hook automático
add_action('publish_post', 'sync_post_to_cdelu');
add_action('publish_page', 'sync_post_to_cdelu');
```

**🎯 Lo que hace automáticamente CdelU:**
- ✅ Marca como oficial (is_oficial=1)
- ✅ Asigna a Matias Moreiraa (created_by=1)
- ✅ Define fecha actual (published_at=NOW())
- ✅ Genera resumen automático (si IA disponible)

### **Python (Django/Flask) - VERSIÓN SIMPLE**
```python
import requests
import os

def sync_to_cdelu(titulo, descripcion, image_url=None, original_url=None):
    """
    Envía una noticia a CdelU. Solo requiere título y descripción.
    El servidor configura automáticamente: oficial, autor, fecha, etc.
    """
    api_url = os.getenv('CDELU_API_URL', 'http://localhost:3001/api/v1/news/external')
    api_key = os.getenv('CDELU_API_KEY')

    headers = {
        'Content-Type': 'application/json',
        'X-API-Key': api_key
    }
    
    # ✅ Datos MÍNIMOS requeridos
    data = {
        'titulo': titulo,
        'descripcion': descripcion
    }
    
    # ⚠️ Campos opcionales
    if image_url:
        data['image_url'] = image_url
    if original_url:
        data['original_url'] = original_url

    response = requests.post(api_url, json=data, headers=headers)

    if response.status_code == 201:
        result = response.json()
        print(f"✅ Noticia creada: ID {result['data']['id']}")
        print(f"   Oficial: {result['data']['is_oficial']}")
        print(f"   Autor: {result['data']['autor']}")
        return True
    else:
        print(f"❌ Error: {response.text}")
        return False

# Uso simple:
sync_to_cdelu(
    titulo="Nueva noticia de votación",
    descripcion="Se implementó un nuevo sistema de votación..."
)
```

### **Ruby (Rails)**
```ruby
class CdeluSyncService
  API_URL = ENV['CDELU_API_URL']
  API_KEY = ENV['CDELU_API_KEY']

  def self.sync_post(post)
    headers = {
      'Content-Type' => 'application/json',
      'X-API-Key' => API_KEY
    }

    payload = {
      titulo: post.title,
      descripcion: post.content,
      image_url: post.featured_image&.url,
      original_url: post.url,
      is_oficial: false
    }

    response = HTTParty.post(API_URL, headers: headers, body: payload.to_json)

    if response.success?
      post.update(cdelu_synced: true)
      Rails.logger.info "Post #{post.id} synced to CdelU"
    else
      Rails.logger.error "Failed to sync post #{post.id}: #{response.body}"
    end
  end
end
```

### **Go**
```go
package main

import (
    "bytes"
    "encoding/json"
    "log"
    "net/http"
    "os"
)

type PostData struct {
    Titulo      string `json:"titulo"`
    Descripcion string `json:"descripcion"`
    ImageURL    string `json:"image_url,omitempty"`
    OriginalURL string `json:"original_url,omitempty"`
    IsOficial   bool   `json:"is_oficial"`
}

func syncToCdelu(post PostData) error {
    apiURL := os.Getenv("CDELU_API_URL")
    apiKey := os.Getenv("CDELU_API_KEY")

    jsonData, err := json.Marshal(post)
    if err != nil {
        return err
    }

    req, err := http.NewRequest("POST", apiURL, bytes.NewBuffer(jsonData))
    if err != nil {
        return err
    }

    req.Header.Set("Content-Type", "application/json")
    req.Header.Set("X-API-Key", apiKey)

    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        return err
    }
    defer resp.Body.Close()

    if resp.StatusCode == 201 {
        log.Printf("Post synced successfully: %s", post.Titulo)
        return nil
    } else {
        log.Printf("Failed to sync post: %d", resp.StatusCode)
        return fmt.Errorf("sync failed with status %d", resp.StatusCode)
    }
}
```

### **C# (.NET)**
```csharp
using System.Net.Http;
using System.Text;
using System.Threading.Tasks;
using Newtonsoft.Json;

public class CdeluSyncService
{
    private readonly string _apiUrl;
    private readonly string _apiKey;
    private readonly HttpClient _httpClient;

    public CdeluSyncService()
    {
        _apiUrl = Environment.GetEnvironmentVariable("CDELU_API_URL");
        _apiKey = Environment.GetEnvironmentVariable("CDELU_API_KEY");
        _httpClient = new HttpClient();
    }

    public async Task<bool> SyncPostAsync(PostData post)
    {
        var json = JsonConvert.SerializeObject(post);
        var content = new StringContent(json, Encoding.UTF8, "application/json");

        _httpClient.DefaultRequestHeaders.Clear();
        _httpClient.DefaultRequestHeaders.Add("X-API-Key", _apiKey);

        var response = await _httpClient.PostAsync(_apiUrl, content);

        if (response.IsSuccessStatusCode)
        {
            // Marcar como sincronizado
            post.CdeluSynced = true;
            await _context.SaveChangesAsync();
            _logger.LogInformation($"Post {post.Id} synced to CdelU");
            return true;
        }
        else
        {
            _logger.LogError($"Failed to sync post {post.Id}: {response.StatusCode}");
            return false;
        }
    }
}

public class PostData
{
    public string Titulo { get; set; }
    public string Descripcion { get; set; }
    public string ImageUrl { get; set; }
    public string OriginalUrl { get; set; }
    public bool IsOficial { get; set; }
}
```
